Set Up
- Install extension Playwright di VSCode
- Install 'npm install playwright'
- npm init playwright@latest
- npm i -D allure-playwright
- npx playwright codegen --viewport-size=1400,575 --channel='chrome' (record dengan Chrome)
- npx playwright test login.spec.ts (untuk running)
- npx playwright test login.spec.ts --project=chromium --headed (untuk running dengan tampilan depan)
